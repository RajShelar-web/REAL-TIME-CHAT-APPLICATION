##Project Overview

A full-stack Real-Time Chat Application built using Node.js, Socket.io, and JavaScript that enables users to send and receive messages instantly. The application demonstrates real-time bidirectional communication between clients and server using WebSockets.

This project showcases strong understanding of frontend development, backend development, and real-time event handling.

## Features

Real-time messaging using Socket.io

Instant message broadcasting

Multiple users can join and chat simultaneously

Live updates without page refresh

Responsive UI using HTML and CSS

Client-server architecture

Event-driven communication
